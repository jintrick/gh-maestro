# スキルアセットの --help 必須

エージェント（orchestrator・ワーカー）が CLI として呼ぶ可能性のあるスキルアセットは、必ず `--help` / `-h` を実装すること。エージェントは使い方を man ページのように `--help` で確認できる前提で動く。

要件:
- `--help` / `-h` で usage を表示し、**終了コード 0** で終わる
- 引数・出力メッセージ（例: `REVIEW_STARTED:<PR>`）・副作用（ロック/ログの場所など）を説明に含める
- 引数不足時は同じ usage を stderr に出し、**終了コード 1** で終わる（help は 0、誤用は 1 と区別する）

「偶然 usage が出る」（引数不足エラーに引数として `--help` が落ちる等）は実装とみなさない。明示的に `--help`/`-h` を分岐すること。

`require.main === module` でない（他スクリプトから require されるだけの）モジュールは対象外。
