# `show_close_tab_button_in_tabs = true`

{{since('nightly')}}

When set to `false`, the close-tab button will not be drawn in tabs when the
fancy tab bar is in use. Default is `true`.

```lua
config.show_close_tab_button_in_tabs = false
```
