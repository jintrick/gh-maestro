# Antigravity Specialized Models

Specialized models used by subagents for non-textual or high-efficiency tasks.

## UI & Design
- **Nano Banana Pro 2**: Generates UI mockups, SVG diagrams, and layout suggestions.
- **Gemini 2.5 Pro UI Checkpoint**: Used by the `browser` subagent for high-fidelity DOM actuation and visual verification.

## Search & Indexing
- **Gemini 2.5 Flash Lite**: Used internally for fast semantic search across large codebases and indexing documentation.
