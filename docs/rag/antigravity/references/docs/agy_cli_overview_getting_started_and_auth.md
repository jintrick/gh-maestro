# Antigravity CLI Overview

Antigravity CLI (agy) is the terminal-based interface for the Antigravity 2.0 AI agentic engine.

## Core Characteristics
- **Binary Name**: `agy` (aliased to `antigravity`).
- **Base Directory**: `~/.gemini/antigravity-cli/`.
- **Primary Interface**: Interactive TUI (Terminal User Interface).
- **Execution Mode**: Local first, with optional cloud-based reasoning models.

## Relationship with Gemini CLI
Antigravity CLI is the official successor to Gemini CLI. It is built in Go (replacing the Node.js core) and introduces a unified plugin system and multi-agent coordination.
