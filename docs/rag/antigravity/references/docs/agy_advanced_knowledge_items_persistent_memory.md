# Antigravity Knowledge Items (KI)

Knowledge Items are a persistent memory system that automatically captures insights, patterns, and solutions from coding sessions.

## Features
- **Auto-Capture**: The system analyzes conversations to extract important information and create or update KIs.
- **RAG Integration**: The agent references KI summaries to ground its answers in previous project context.
- **Context Retention**: Memory persists across sessions, preventing the need to re-explain project-specific logic.

## Usage
Agents automatically use Knowledge Items when relevant. High-relevance items trigger a deeper fetch of detailed artifacts (docs, code snippets) to ensure accuracy.
