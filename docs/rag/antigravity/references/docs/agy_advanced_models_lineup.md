# Antigravity Model Lineup

Antigravity provides access to the leading reasoning models on the Gemini Enterprise platform.

## Available Models
- **Gemini 3.5 Flash**: Optimized for speed and high-volume tasks.
- **Gemini 3.1 Pro (High/Low)**: The default for complex reasoning and large-scale coding.
- **Gemini 3 Flash**: Standard high-speed coding assistant.
- **Claude 4.6 Sonnet/Opus (Thinking)**: Integrated via Vertex AI for diverse reasoning styles.
- **GPT-OSS-120b**: Open-source alternative for specific logic tasks.
- **Nano Banana 2**: Specialized for UI mockups, diagrams, and image generation.

## Switching Models
Use the `/model` command in the TUI to switch at any time. Changes are "sticky" within a turn.
