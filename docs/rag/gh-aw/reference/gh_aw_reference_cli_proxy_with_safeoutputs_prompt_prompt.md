<gh-cli>
The `gh` CLI is pre-authenticated. You SHOULD use `gh` commands for all GitHub reads: listing and searching issues, pull requests, discussions, labels, milestones; reading workflow runs, jobs, and artifacts; accessing repository contents, code, and metadata. Use safeoutputs tools for GitHub writes and completion signaling. No GitHub MCP server is available; `gh` is your only way to read from GitHub.
</gh-cli>
